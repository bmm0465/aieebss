AIEEBSS란?

AIEEBSS (AI for Elementary English Basic Skills Support)는 AI 기반 영어 읽기 평가 시스템입니다. 음성 인식과 인공지능을 활용하여 학생들의 영어 읽기 능력을 자동으로 평가하고 분석합니다.

✨ 주요 특징

  • 6가지 DIBELS 테스트: 알파벳부터 이해력까지 단계별 평가
  • AI 음성 인식: OpenAI Whisper로 정확한 음성 전사
  • 자동 채점: GPT-4o-mini를 활용한 지능형 평가
  • 교사 대시보드: 담당 학생들의 결과를 한눈에 확인
  • EFL 환경 최적화: 한국 학생 발음 특성 반영

💡 시스템 요구사항

  • 최신 웹 브라우저 (Chrome, Edge, Safari 권장)
  • 마이크 권한 허용
  • 안정적인 인터넷 연결
  • 학생 계정 (교사가 발급)

👨‍🎓 학생 가이드

1. 로그인하기

1 웹사이트 접속

선생님이 제공한 웹사이트 주소로 접속합니다.

2 이메일과 비밀번호 입력

선생님이 발급해준 이메일 주소와 비밀번호를 입력합니다.

3 로비 페이지 확인

로그인 성공 시 "달빛 마법학교" 로비 페이지가 나타납니다. 여섯 가지 시험 과목을 확인할 수 있습니다.

⚠️ 주의사항
  • 비밀번호는 다른 사람과 공유하지 마세요
  • 로그아웃 후에는 반드시 브라우저를 닫아주세요

2. 테스트 시작하기

1 테스트 선택

로비 페이지에서 원하는 테스트를 클릭하거나, 🎯 첫 번째 시험 시작하기 버튼을 클릭합니다.

2 마이크 권한 허용

브라우저에서 마이크 사용 권한을 요청하면 "허용"을 선택합니다. 필수

3 시험 시작

시험 시작하기 버튼을 클릭합니다. 마이크가 준비되지 않았다면 "마이크 준비 중..." 메시지가 표시됩니다.

3. 테스트 진행 방법

🎤 음성 녹음 테스트 (p1, p4)

예시: 알파벳 읽기, 단어 읽기

  1. 화면에 나타나는 문자나 단어를 확인합니다
  2. 녹음하기 버튼을 클릭합니다
  3. 명확하게 읽어줍니다 (최대 5초)
  4. 녹음 끝내기 버튼을 클릭합니다
  5. 자동으로 다음 문제로 넘어갑니다
💡 팁: 스페이스바나 엔터키로도 녹음을 시작/중지할 수 있습니다.

🎧 듣기 테스트 (p2, p3)

예시: 소리 구분, 강세 패턴

  1. 🔊 재생 버튼을 클릭하여 오디오를 듣습니다
  2. 들려준 내용을 기억합니다
  3. 정답이라고 생각하는 선택지를 클릭합니다
  4. 자동으로 다음 문제로 넘어갑니다

🖼️ 선택형 테스트 (p5, p6)

예시: 그림 선택, 이해력 문제

  1. 문제를 읽거나 오디오를 듣습니다
  2. 정답이라고 생각하는 그림이나 선택지를 클릭합니다
  3. 자동으로 다음 문제로 넘어갑니다

⏱️ 시간 제한

  • p1~p5: 각 1분 (60초)
  • p6: 3분 (180초)
  • 시간이 끝나면 자동으로 시험이 종료됩니다
⚠️ 테스트 중 주의사항
  • 조용한 환경에서 테스트를 진행하세요
  • 마이크에 가까이 말하되, 너무 크게 말하지 마세요
  • 문제를 넘어가고 싶다면 ⏭️ 넘어가기 버튼을 사용하세요
  • 테스트 중에는 페이지를 새로고침하거나 닫지 마세요

4. 결과 확인하기

1 테스트 완료

모든 테스트를 완료하면 "시험 종료!" 메시지가 나타납니다.

2 결과 페이지 접속

로비 페이지에서 📊 이전 평가 결과 보기 버튼을 클릭합니다.

3 세션별 결과 확인

각 테스트 세션별로 정확도, 정답/오답 현황을 확인할 수 있습니다.

✅ 완료!

테스트 결과는 자동으로 저장되며, 선생님도 확인할 수 있습니다.

👩‍🏫 교사 가이드

1. 교사 대시보드 접속

1 로그인

교사 계정으로 로그인합니다.

2 대시보드 버튼 클릭

로비 페이지에서 🎓 교사 관리 대시보드 버튼을 클릭합니다.

💡 참고: 교사 계정은 관리자가 설정합니다. 교사 권한이 없다면 관리자에게 문의하세요.

2. 대시보드 주요 기능

📊 통계 요약

대시보드 상단에 다음 통계가 표시됩니다:

  • 총 학생 수
  • 담당 반 수
  • 전체 테스트 수
  • 평균 정확도

👥 반별 학생 목록

담당 학생들이 반별로 그룹화되어 표시됩니다:

  • 학생 이름 및 번호
  • 완료한 테스트 수
  • 완료율 (%)
  • 평균 정확도 (%)
  • 마지막 테스트 날짜

3. 학생 상세 결과 확인

1 학생 카드 클릭

대시보드에서 확인하고 싶은 학생의 카드를 클릭합니다.

2 상세 결과 페이지

학생의 상세 평가 결과 페이지가 열립니다.

📈 제공되는 정보

  • 전체 평가 현황: 6가지 테스트별 정확도 및 정답/오답 수
  • 최근 테스트 결과: 개별 문제별 정답/오답 상세 내역
  • 시각화 차트: 막대 차트, 레이더 차트로 역량 시각화
  • AI 평가 코멘트: 자동 생성된 종합 평가 및 개선 제안

4. 교사 대시보드 추가 기능

📋 평가 문항 및 정답 확인

대시보드 상단의 📋 평가 문항 및 정답 확인 버튼을 클릭하면 각 테스트의 문항과 정답을 확인할 수 있습니다.

🤖 AI 문항 생성기

🤖 AI 문항 생성기 버튼을 통해 새로운 평가 문항을 AI로 생성할 수 있습니다.

📚 교육과정 데이터

📚 교육과정 데이터 버튼으로 교육과정 관련 데이터를 관리할 수 있습니다.

💡 활용 팁

  • 정기적으로 대시보드를 확인하여 학생들의 진도를 파악하세요
  • 평균 정확도가 낮은 학생에게는 추가 지도가 필요할 수 있습니다
  • 완료율이 낮은 학생에게는 테스트 완료를 독려하세요
  • AI 평가 코멘트를 참고하여 학생별 맞춤 지도 계획을 수립하세요

📚 6가지 테스트 종류

테스트 이름 평가 내용 시간 방식
p1 고대 룬 문자 해독
(Letter Naming)		 알파벳 대소문자 이름 인식 속도 1분 음성 녹음
p2 소리의 원소 분리
(Segmental Phoneme)		 비슷한 소리를 듣고 구분하는 능력 1분 듣기 + 선택
p3 마법 리듬 패턴
(Suprasegmental Phoneme)		 단어의 강세 패턴을 듣고 선택 1분 듣기 + 선택
p4 마법 주문 읽기
(Phonics)		 무의미 단어, 실제 단어, 문장 읽기 1분 음성 녹음
p5 마법서 그림 해석
(Vocabulary)		 단어나 구문의 의미를 그림으로 선택 1분 듣기/읽기 + 선택
p6 고대 전설 이해
(Comprehension)		 이야기를 듣고/읽고 질문에 답하기 3분 듣기/읽기 + 선택

🎯 테스트 순서

테스트는 p1부터 p6까지 순서대로 진행하는 것을 권장하지만, 원하는 테스트를 선택하여 진행할 수도 있습니다.

📖 DIBELS 기반 평가

모든 테스트는 DIBELS 8th Edition을 기반으로 하며, 영어 읽기 능력을 단계별로 종합적으로 평가합니다.

❓ 자주 묻는 질문 (FAQ)

Q1. 마이크가 작동하지 않아요

해결 방법:
  1. 브라우저 설정에서 마이크 권한을 확인하세요
  2. 다른 프로그램이 마이크를 사용 중인지 확인하세요
  3. 브라우저를 재시작한 후 다시 시도하세요
  4. Chrome 또는 Edge 브라우저 사용을 권장합니다

Q2. 테스트 중 오디오가 재생되지 않아요

해결 방법:
  1. 스피커 볼륨을 확인하세요
  2. 브라우저에서 오디오 자동 재생이 허용되어 있는지 확인하세요
  3. 다른 탭에서 오디오가 재생 중인지 확인하세요
  4. 페이지를 새로고침한 후 다시 시도하세요

Q3. 테스트 결과가 저장되지 않았어요

확인 사항:
  1. 인터넷 연결 상태를 확인하세요
  2. 테스트를 완료했는지 확인하세요 (시간이 끝나거나 모든 문제를 풀었는지)
  3. 결과 페이지에서 세션 목록을 확인하세요
  4. 문제가 계속되면 선생님에게 문의하세요

Q4. 교사 대시보드에 학생이 보이지 않아요

확인 사항:
  1. 교사 계정으로 로그인했는지 확인하세요
  2. 학생이 테스트를 완료했는지 확인하세요
  3. 학생이 올바르게 배정되었는지 관리자에게 확인하세요

Q5. 테스트를 중간에 중단할 수 있나요?

네, 가능합니다. 테스트 중에 🏠 홈으로 가기 버튼을 클릭하면 로비로 돌아갈 수 있습니다. 다만, 중단한 테스트의 결과는 저장되지 않을 수 있습니다.

Q6. 같은 테스트를 여러 번 할 수 있나요?

네, 가능합니다. 같은 테스트를 여러 번 진행할 수 있으며, 각 시도는 별도의 세션으로 저장됩니다. 선생님은 모든 세션의 결과를 확인할 수 있습니다.

Q7. 테스트 결과는 어떻게 확인하나요?

학생: 로비 페이지의 "이전 평가 결과 보기" 버튼 클릭
교사: 대시보드에서 학생 카드 클릭하여 상세 결과 확인

⚠️ 추가 도움이 필요하신가요?

문제가 해결되지 않거나 추가 도움이 필요하시면 관리자나 시스템 담당자에게 문의하세요.
