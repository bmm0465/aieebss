# 파일럿 테스트 피드백 가이드

## 📋 테스트 목적
초등 영어 기초 학력 진단 평가 시스템의 파일럿 테스트를 통해 사용자 경험, 기술적 안정성, 평가 내용의 적절성을 검증합니다.

---

## 👥 학생 대상 피드백 항목

### 1. 사용 편의성 (Usability)

#### 📱 화면 및 인터페이스
- [ ] **화면 크기와 레이아웃**: 화면이 보기 편했나요? 텍스트나 버튼이 너무 작거나 크지 않았나요?
- [ ] **색상 및 대비**: 색상 구분이 명확했나요? (특히 정답/오답 표시)
- [ ] **버튼 위치**: 재생 버튼, 다음 버튼 등의 위치가 적절했나요?
- [ ] **한글/영어 표시**: 한글 안내문이 이해하기 쉬웠나요?

#### 🔊 음성 및 오디오
- [ ] **음성 속도**: 음성이 너무 빠르거나 느리지 않았나요?
- [ ] **음성 명확도**: 발음이 명확하게 들렸나요?
- [ ] **재생 기능**: "다시 듣기" 버튼이 잘 작동했나요?
- [ ] **대화 음성** (6교시): 두 화자의 음성이 구분되게 들렸나요?
- [ ] **음성 반복**: 음성이 2회 반복되는 것이 적절했나요?

### 2. 평가 내용 및 난이도

#### 📚 문항 난이도
- [ ] **1교시 (알파벳)**: 알파벳 순서와 난이도가 적절했나요?
- [ ] **2교시 (음소)**: 단어 발음이 너무 어렵지 않았나요?
- [ ] **3교시 (초분절음)**: 강세 구분이 명확했나요?
- [ ] **4교시 (파닉스)**: 문장 읽기가 적절한 난이도였나요?
- [ ] **5교시 (어휘)**: 이미지와 단어 매칭이 이해하기 쉬웠나요?
- [ ] **6교시 (이해력)**: 대화 내용이 이해하기 쉬웠나요?

#### 🎯 문항 구성
- [ ] **문항 수**: 각 교시별 문항 수(시간)가 적절했나요?
- [ ] **보기 개수**: 선택지가 너무 많거나 적지 않았나요?
- [ ] **타임리밋**: 제한 시간이 적절했나요?

### 3. 기술적 문제

#### ⚠️ 오류 및 문제점

- [ ] **오디오 재생**: 음성 파일이 재생되지 않는 경우가 있었나요?
- [ ] **이미지 로딩**: 이미지가 제대로 표시되었나요?
- [ ] **버튼 작동**: 버튼이 클릭되지 않거나 멈춘 적이 있었나요?
- [ ] **인터넷 연결**: 인터넷이 끊기거나 느려진 적이 있었나요?

## 📊 피드백 수집 방법

### 1. 설문 조사
- **학생**: 구두 또는 간단한 체크리스트 (교사가 대신 작성)
- **교사**: 온라인 설문 또는 서면 설문

### 2. 관찰 기록
- **관찰 일지**: 교사가 평가 진행 중 관찰한 내용 기록
- **비디오 녹화** (동의 하에): 학생의 평가 진행 과정 녹화 후 분석

### 3. 인터뷰
- **학생 인터뷰**: 평가 후 간단한 질문 (10분 내외)
  - 가장 어려웠던 부분은?
  - 가장 쉬웠던 부분은?
  - 개선했으면 하는 점은?
- **교사 인터뷰**: 평가 결과 분석 및 종합 의견 (20-30분)

### 4. 데이터 분석
- **시스템 로그**: 오류 발생 빈도, 응답 시간 등
- **결과 분석**: 정답률, 완료율, 시간 분포 등

## 📅 테스트 일정 예시

### 1주차: 사전 준비
- 교사 대상 평가 방법 안내
- 테스트 환경 점검
- 학생 계정 생성

### 2주차: 파일럿 테스트
- **Day 1**: 학생 3-5명 평가 (약 30-40분/명)
- **Day 2**: 학생 3-5명 평가
- **Day 3**: 평가 결과 검토 및 초기 피드백 수집

### 3주차: 피드백 분석
- 설문 조사 수집
- 인터뷰 진행
- 데이터 분석
- 개선 사항 도출

## 📌 추가 권장 사항

### 테스트 전
- 학생들에게 평가 목적과 방법을 충분히 설명
- 교사가 먼저 평가를 경험해볼 수 있도록 지원
- 테스트 환경(인터넷, 헤드셋 등) 사전 점검

### 테스트 중
- 학생이 불편해하거나 막히는 상황을 즉시 기록
- 기술적 문제 발생 시 즉시 중단하고 기록
- 학생의 자연스러운 반응 관찰

### 테스트 후
- 결과에 대한 학생 피드백 즉시 수집
- 교사 종합 의견 수집
- 시스템 로그 및 데이터 분석
